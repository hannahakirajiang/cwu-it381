This page has been created by adding data-rel="dialog" to the link on the parent page:

<div data-role="page" id="dialogpage" data-rel="dialog">

Clicking on any link in this dialog will close it. This button is linked back to the main page:
Return

A cancel button is created by linking back to the page that called this dialog and adding data-rel="back" to the link.

<a href="#" data-role="button" data-rel="back"> Cancel</a>
